About Me

I am a fullstack software engineer who is passionate about writing and consistently learning more about code. I really enjoy problem solving, and being able to constantly think about code even while being away from my computer. I love the idea of being part of a development team that create innovative and functional web applications, and being able to physically see consumers making use of a product that I took part in designing.

Prior to coding, I have an extensive background in the performance arts. I majored as a Classical Guitar Performance major for my Bachelor's and Master's degrees, and I am the recipient of multiple competition awards and certifications.

I have experience coding up projects in Javascript and Python. I have used frameworks and other technologies such as React/Redux, Express, Flask, Sequelize, and SQLAlchemy for these projects.

Projects

  • Cloneagram

    Cloneagram is an Instagram clone created using Flask and React. This application gives users the ability to create posts, comment on posts, like posts, and connect with one another in real time through direct messaging. Image uploads and the real time chat function are made possibly with the implementation of AWS S3 and Socket.io, respectively.
    GitHub Live

  • Harmonious Voices

    Harmonious Voices is an application that allows users to create communities to connect with one another. Being that it is cloned from the popular Discord app, users are able to achieve this by creating servers and channels where they can chat with multiple different users. Users are also given the option to privately message one another through direct messaging.
    GitHub Live

  • Groundbnb

    Groundbnb is an application built on the PERN stack. This Airbnb clone comes with features for users to create listings that other users can book. With full CRUD functionality, users are given the option to edit and delete their bookings and/or listings as well.
    GitHub Live

  • Betternote

    Betternote is an Evernote clone with features for creating and editing notes. For better organization, users can collect their notes into separate notebooks and also move them around whenever needed.
    GitHub Live

  • The Muse Duo

    Personal site of the guitar and piano duo the Muse Duo. Created fully using React and node packages such as Swiper and React Video that allow for a greater user experience when navigating through the page. This site is used to promote the duo and spread awareness of their art.
